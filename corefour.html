Variables A variable stores a changeable value that can be used elsewhere in the code, such as integers and floats, strings, booleans, lists, and any other relevant information.
Loops A loop allows a section of code to repeat for as long as necessary, while its specified conditions are met.
Conditionals A conditional runs a certain section of code based on whether a specified condition is met or not. It is usually in the form of if-then statements, which check for the condition.
Functions A function defines a section of code that can be called upon many times to perform a specific task, which condenses the overall code.
